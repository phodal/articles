为什么会有这个Repo？
---

如你所见：我已经有了自己的博客：[https://www.phodal.com/](https://www.phodal.com/)，也有自己不同平台的专栏，为什么还有这样的一个项目？

> 原因主要是因为有一些文章不太适合发在博客上，博客上的内容主要以技术为主——依赖于SEO。并且有一些文章主要是以图片为主，这些文章难以直接复制到各个平台上使用。

项目后期会以网页的形式出现——编辑-分布-开发分离。
